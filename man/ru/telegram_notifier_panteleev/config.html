Данный модуль предназначен для отправки PUSH-уведомлений в Telegram (в будущем возможно другие мессенджеры).

Описание опций

  • Включить PUSH-уведомления: активирует или деактивирует отправку сообщений в Telegram
  • API бота: в данной версии модуля не используется
  • Код бота Telegram: код который отправил бот после того, как вы нажили "Старт"
  • URL corezoid.com: URL к вебхуку corezoid.com, куда UMI.CMS будет отправлять запросы на сообщения.
